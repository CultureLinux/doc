    Prochain kernel au boot

    grub2-editenv list

    Forcer le prochain kernel au boot

    grub2-reboot "Rocky Linux (5.14.0-503.14.1.el9_5.x86_64) 9.5 (Blue Onyx)"

    grub2-set-default "Rocky Linux (5.14.0-503.33.1.el9_5.x86_64) 9.5 (Blue Onyx)"
